⚡ What are JavaScript Functions?

A function is a reusable block of code that performs a specific task. Instead of writing the same code again and again, you can create a function and call it whenever needed.

📌 Function Declaration

function greet(){

    console.log("Hello World");

}

greet();

The function runs only when it is called.

📌 Function with Parameters

function greet(name){

    console.log("Hello " + name);

}

greet("Shreya");

Parameters allow you to pass values into a function.

📌 Return Statement

function add(a,b){

    return a+b;

}

console.log(add(5,3));

The return statement sends a value back from the function.

📌 Arrow Function (ES6)

const add = (a,b) => {

    return a+b;

};

Arrow functions provide a shorter way to write functions.

⭐ Best Practices

  • Keep functions short and simple.
  • Use meaningful function names.
  • Reuse functions whenever possible.
  • Avoid repeating code.

❓ Frequently Asked Questions

1. Why are functions used?

Functions reduce code repetition and make programs easier to maintain.

2. What is a parameter?

A parameter is a value passed into a function.

3. What does return do?

It sends a value back from the function.
