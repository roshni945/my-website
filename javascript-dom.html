📖 What is DOM?

The DOM (Document Object Model) is a programming interface that allows JavaScript to interact with HTML documents.

Using DOM, JavaScript can change text, styles, attributes, add new elements, and remove existing elements without reloading the page.

🌳 HTML DOM Tree

The browser converts HTML into a tree structure called the DOM Tree. Every HTML tag becomes a node. 

HTML
 |
 └── body
      |
      ├── h1
      ├── p
      └── button

🔍 Selecting Elements

getElementById()

document.getElementById("demo");
Selects an element using its ID.

querySelector()

document.querySelector("h1");
Selects the first matching element.

querySelectorAll()

document.querySelectorAll("p");
Selects all matching elements.

✏️ Changing Content

document.getElementById("demo").innerHTML = "Welcome!";
Output:
Welcome! 
document.getElementById("demo").textContent = "Hello";
Output:
Hello

🎨 Changing CSS

document.getElementById("demo").style.color="red";
Changes the text color to red.

📝 Changing Attributes

document.getElementById("img")
.setAttribute("width","200");
Changes image width.

➕ Creating Elements

let p = document.createElement("p");

p.textContent="New Paragraph";

document.body.appendChild(p);
Creates and adds a new paragraph.

🗑️ Removing Elements

document.getElementById("demo").remove();
Removes the selected element.

🛠️ Common DOM Methods

Method Description
getElementById() Selects element by ID.
querySelector() Selects first matching element.
querySelectorAll() Selects all matching elements.
innerHTML Changes HTML content.
textContent Changes text only.
style Changes CSS styles.
setAttribute() Changes attributes.
createElement() Creates new HTML elements.
appendChild() Adds new child element.
remove() Deletes an element.

💡 Did You Know?

JavaScript can update a complete webpage instantly using DOM without refreshing the page.

⭐ Key Points

  • DOM connects JavaScript with HTML.
  • Every HTML element becomes a node.
  • DOM allows creating, changing and deleting elements.
  • DOM makes websites interactive.

⭐ Best Practices

  • Use meaningful IDs and classes.
  • Keep JavaScript separate from HTML.
  • Use querySelector() for flexible selection.
  • Avoid unnecessary DOM changes.

⚠️ Common Mistakes

  • Wrong element ID.
  • Running JavaScript before HTML loads.
  • Misspelling DOM methods.
  • Using innerHTML unnecessarily.

📖 JavaScript DOM Glossary

DOM
Document Object Model. It represents HTML as objects that JavaScript can access.
Node
Every item in the DOM tree is called a node.
Element
An HTML tag represented as an object in DOM.
DOM Tree
A tree structure created by the browser from HTML elements.
Document
The complete HTML page loaded in the browser.
Selector
A pattern used to select HTML elements.
innerHTML
A property used to change HTML content inside an element.
textContent
A property used to change only text content.
Attribute
Extra information inside HTML tags like id, class, and src.
Event
An action like click, hover, or keyboard input handled by JavaScript.

❓ Frequently Asked Questions (FAQs)

1. What is DOM in JavaScript?

DOM (Document Object Model) is a programming interface that allows JavaScript to access and modify HTML elements on a webpage.

2. Why is DOM important in JavaScript?

DOM makes webpages interactive by allowing JavaScript to change content, styles, and elements dynamically.

3. How does the browser create DOM?

The browser converts HTML code into a tree structure called the DOM Tree. Each HTML element becomes a node.

4. What is the difference between HTML and DOM?

HTML is the original structure of a webpage, while DOM is the browser-created object representation that JavaScript can manipulate.

5. What is getElementById()?

getElementById() is a DOM method used to select an HTML element using its ID.

6. What is querySelector()?

querySelector() selects the first element that matches a CSS selector.

7. Difference between querySelector() and querySelectorAll()?

querySelector() returns the first matching element, while querySelectorAll() returns all matching elements.

8. Can JavaScript change CSS using DOM?

Yes, JavaScript can change CSS properties using the style property.

9. How can we create a new element using DOM?

We can create new elements using createElement() and add them using appendChild().

10. How can we remove an element using DOM?

The remove() method is used to delete an element from the webpage.
