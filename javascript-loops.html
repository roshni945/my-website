🔁 What are JavaScript Loops?

Loops allow JavaScript to execute the same block of code multiple times until a condition becomes false.

📌 for Loop

for(let i = 1; i <= 5; i++){
    console.log(i);
}

A for loop is used when you know how many times the loop should run.

📌 while Loop

let i = 1;

while(i <= 5){
    console.log(i);
    i++;
}

A while loop runs as long as its condition is true.

📌 do...while Loop

let i = 1;

do{
    console.log(i);
    i++;
}while(i <= 5);

A do...while loop always executes at least once.

📌 break Statement

for(let i=1;i<=10;i++){

    if(i==5){
        break;
    }

    console.log(i);

}

The break statement immediately stops the loop.

📌 continue Statement

for(let i=1;i<=5;i++){

    if(i==3){
        continue;
    }

    console.log(i);

}

The continue statement skips the current iteration.

⭐ Best Practices

  • Use for loops when the number of iterations is known.
  • Use while loops when the condition controls the loop.
  • Avoid infinite loops.
  • Keep loop code simple and readable.

❓ Frequently Asked Questions

1. Which loop is most commonly used?

The for loop is the most commonly used loop.

2. What does break do?

It immediately stops the loop.

3. What does continue do?

It skips the current iteration and continues with the next one.
