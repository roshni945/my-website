🔄 DOM Traversing in JavaScript

DOM Traversing means moving between HTML elements in the DOM Tree. JavaScript allows us to access parent, child, and sibling elements easily.

📘 Definition

DOM Traversing is the process of navigating through DOM elements using properties like parentElement, children, firstElementChild, and nextElementSibling.

💡 Did You Know?

Every HTML element has relationships with other elements. An element can have a parent, children, and siblings in the DOM Tree.

👨‍👩‍👦 Access Parent Element

The parentElement property returns the parent HTML element of the selected element.

Example: parentElement


<div id="box">

    <p id="text">
        Hello DOM
    </p>

</div>


<script>

let text = document.getElementById("text");

console.log(text.parentElement);

</script>

✅ Output

Returns the parent div element.

👶 Access Child Elements

The children property returns all child elements inside a selected element.

Example: children


<div id="parent">

    <p>HTML</p>

    <p>CSS</p>

    <p>JavaScript</p>

</div>


<script>

let parent = document.getElementById("parent");

console.log(parent.children);

</script>

✅ Output

HTMLCollection(3)

🌳 childNodes

The childNodes property returns all child nodes including text nodes, comments, and elements.

Example: childNodes



let box = document.querySelector(".box");

console.log(box.childNodes);

✅ Output

Returns all nodes inside the element.

➡️ First and Last Child

JavaScript provides properties to access the first and last child element.

Example: firstElementChild & lastElementChild



let list = document.querySelector("ul");


console.log(list.firstElementChild);


console.log(list.lastElementChild);

✅ Output

Returns the first and last element inside the list.

↔️ Sibling Elements

Sibling elements are elements that have the same parent element. JavaScript provides methods to move between siblings.

Example: nextElementSibling



let first = document.querySelector("p");


console.log(first.nextElementSibling);

✅ Output

Returns the next paragraph element.

🌍 Real-Life Example

Imagine a shopping website product card:

  • 🛒 Product Box → Parent Element
  • 🏷️ Product Name & Price → Child Elements
  • ➡️ Next Product Card → Sibling Element

JavaScript can move between these elements using DOM Traversing methods.

⭐ Key Points

  • ✅ parentElement finds the parent element.
  • ✅ children returns only HTML child elements.
  • ✅ childNodes returns all nodes.
  • ✅ firstElementChild selects the first child element.
  • ✅ nextElementSibling moves to the next sibling.

🧩 Creating Dynamic Content in JavaScript

JavaScript DOM allows us to create new HTML elements dynamically without writing them directly in the HTML file. We can create elements, add text, and insert them anywhere on the webpage.

📘 Definition

Creating dynamic content means generating new HTML elements using JavaScript while the webpage is running. Methods like createElement() and append() help developers build interactive webpages.

💡 Did You Know?

Modern websites like shopping carts, comment sections, and social media feeds use dynamic DOM creation to add new content instantly.

➕ createElement() Method

The createElement() method creates a new HTML element using JavaScript.

Example 1: Creating a New Paragraph



<script>


let paragraph = document.createElement("p");


paragraph.innerHTML = "Hello, JavaScript DOM";


document.body.appendChild(paragraph);


</script>

✅ Output

A new paragraph is added to the webpage.

📝 createTextNode() Method

The createTextNode() method creates text content separately and adds it to an element.

Example: createTextNode()




let heading = document.createElement("h2");


let text = document.createTextNode(
"Welcome to Roshni Code Charm"
);


heading.appendChild(text);


document.body.appendChild(heading);

✅ Output

A new heading with text is created.

📌 append() Method

The append() method adds elements or text at the end of a parent element.

Example: append()



let box = document.getElementById("box");


let item = document.createElement("p");


item.textContent = "New Item Added";


box.append(item);

✅ Output

New item appears at the end of the box.

⬆️ prepend() Method

The prepend() method adds content at the beginning of a parent element.

Example: prepend()




let list = document.querySelector("ul");


let firstItem = document.createElement("li");


firstItem.textContent = "First Item";


list.prepend(firstItem);

✅ Output

New item is added at the top of the list.

↔️ insertBefore() Method

The insertBefore() method inserts a new element before an existing element.

Example: insertBefore()




let parent = document.getElementById("list");


let newItem = document.createElement("li");


newItem.textContent = "New Item";


let oldItem = parent.children[1];


parent.insertBefore(newItem, oldItem);

✅ Output

New item is inserted before the second item.

🌍 Real-Life Example

Think about a social media app:

  • 💬 Adding a new comment → createElement()
  • ❤️ Adding a new notification → append()
  • 📌 Pinning important content at top → prepend()
  • 📝 Adding items in a specific position → insertBefore()

⭐ Key Points

  • ✅ createElement() creates new HTML elements.
  • ✅ createTextNode() creates text separately.
  • ✅ append() adds content at the end.
  • ✅ prepend() adds content at the beginning.
  • ✅ insertBefore() inserts content before another element.
  • ✅ Dynamic DOM creation makes websites interactive.

🔁 Replacing Elements in JavaScript

JavaScript DOM provides methods to replace existing HTML elements with new elements dynamically. This is useful when we want to update content without refreshing the webpage.

📘 Definition

Replacing elements means removing an existing element and placing a new element in its position using DOM methods like replaceWith() and replaceChild().

💡 Did You Know?

Many modern websites update sections dynamically using element replacement. For example, updating a shopping cart or changing a profile section.

🔄 replaceWith() Method

The replaceWith() method replaces an existing element with a new element.

Example: replaceWith()



<h2 id="title">
Old Heading
</h2>


<script>


let heading = document.getElementById("title");


let newHeading = document.createElement("h2");


newHeading.textContent = "New Heading";


heading.replaceWith(newHeading);


</script>

✅ Output

Old Heading is replaced with New Heading.

🔁 replaceChild() Method

The replaceChild() method replaces one child element with another child element inside a parent element.

Example: replaceChild()




let parent = document.getElementById("box");


let oldElement = document.getElementById("old");


let newElement = document.createElement("p");


newElement.textContent = "New Content";


parent.replaceChild(newElement, oldElement);

✅ Output

Old child element is replaced with a new element.

❌ Removing Elements in JavaScript

JavaScript allows us to remove HTML elements dynamically from a webpage. This helps create interactive applications where content can be deleted.

🗑️ remove() Method

The remove() method directly removes the selected element from the DOM.

Example: remove()




let box = document.getElementById("box");


box.remove();

✅ Output

The complete box element is removed from the webpage.

🗑️ removeChild() Method

The removeChild() method removes a specific child element from a parent element.

Example: removeChild()




let parent = document.getElementById("list");


let item = document.getElementById("item1");


parent.removeChild(item);

✅ Output

Selected child element is removed.

🌍 Real-Life Example

Consider an online shopping website:

  • 🛒 Removing a product from cart → remove()
  • 🔄 Updating product details → replaceWith()
  • ❌ Deleting a comment → removeChild()
  • 📝 Updating a message → replaceChild()

⭐ Key Points

  • ✅ replaceWith() replaces an element directly.
  • ✅ replaceChild() replaces a child element inside a parent.
  • ✅ remove() removes an element completely.
  • ✅ removeChild() removes a specific child element.
  • ✅ These methods help create dynamic webpages.

📋 Working with Forms using DOM

JavaScript DOM allows us to access and control form elements like input fields, checkboxes, buttons, and select boxes dynamically.

📘 Definition

Working with forms means using JavaScript to read user input, change form values, validate data, and control form elements without reloading the webpage.

💡 Did You Know?

Login forms, registration forms, search boxes, and online checkout forms all use DOM manipulation to collect and process user data.

📝 value Property

The value property is used to get or change the value of an input element.

Example: Getting Input Value



<input id="username" value="Roshni">


<script>


let name = document.getElementById("username").value;


console.log(name);


</script>

✅ Output

Roshni

☑️ checked Property

The checked property is used with checkbox and radio buttons to check whether they are selected or not.

Example: Checkbox Checked




<input type="checkbox" id="agree">


<script>


let check = document.getElementById("agree").checked;


console.log(check);


</script>

✅ Output

true / false

🚫 disabled Property

The disabled property is used to enable or disable HTML elements like buttons and input fields.

Example: Disable Button




let button = document.getElementById("btn");


button.disabled = true;

✅ Output

Button becomes disabled.

🎨 Advanced Class Manipulation

The classList property allows JavaScript to add, remove, check, and toggle CSS classes dynamically.

📘 Definition

classList is a DOM property that provides methods to manage CSS classes of HTML elements.

🔍 classList.contains()

The contains() method checks whether an element has a specific class or not.

Example: contains()




let box = document.querySelector(".box");


console.log(
box.classList.contains("active")
);

✅ Output

true or false

🔄 classList.toggle()

The toggle() method adds a class if it does not exist and removes it if it already exists.

Example: Dark Mode Toggle




function darkMode(){


document.body.classList.toggle("dark");


}

✅ Output

Dark mode class is added or removed.

🌍 Real-Life Example

  • 🔐 Login form → Getting username and password values
  • ☑️ Accept terms checkbox → Using checked property
  • 🌙 Dark mode button → Using classList.toggle()
  • 🚫 Disable submit button → Using disabled property

⭐ Key Points

  • ✅ value gets or changes input values.
  • ✅ checked works with checkbox and radio buttons.
  • ✅ disabled controls element availability.
  • ✅ classList.contains() checks classes.
  • ✅ classList.toggle() adds or removes classes.

⚡ DOM Performance Tips

DOM manipulation is powerful, but too many unnecessary DOM updates can slow down a webpage. Developers should use efficient techniques to improve performance.

📘 Definition

DOM Performance means optimizing the way JavaScript interacts with HTML elements so websites load faster and work smoothly.

💡 Did You Know?

Every time JavaScript changes the DOM, the browser may need to recalculate styles and update the page layout. Too many updates can reduce performance.

🚀 1. Store DOM Selectors in Variables

Instead of selecting the same element multiple times, store it in a variable and reuse it.

Example



// Better Approach

let heading = document.getElementById("title");


heading.style.color = "blue";

heading.style.fontSize = "30px";

🚀 2. Use DocumentFragment()

DocumentFragment allows you to create multiple elements in memory before adding them to the webpage.

Example: DocumentFragment()




let fragment = document.createDocumentFragment();


let item = document.createElement("p");


item.textContent = "New Item";


fragment.appendChild(item);


document.body.appendChild(fragment);

✅ Output

Content is added efficiently to the page.

🚀 3. Avoid Unnecessary DOM Changes

Updating the DOM again and again can decrease performance. Try to update multiple changes together.

🚀 4. Use Modern DOM Methods

  • ✅ Use append() instead of older methods when possible.
  • ✅ Use querySelector() for flexible selection.
  • ✅ Use classList for changing classes.
  • ✅ Avoid creating unnecessary elements.

🌍 Real-Life Example

Large websites like shopping platforms and social media apps use optimized DOM techniques to display thousands of products, posts, and comments smoothly.

⭐ Key Points

  • ✅ Store frequently used DOM elements in variables.
  • ✅ Use DocumentFragment for multiple elements.
  • ✅ Reduce unnecessary DOM updates.
  • ✅ Efficient DOM coding improves website speed.
  • ✅ Performance optimization is important for large applications.

💼 Interview Questions

Common DOM Interview Questions

  • Q1: What is DOM?
    Ans: DOM is a programming interface that represents HTML as objects.

  • Q2: Difference between DOM and BOM?
    Ans: DOM handles webpage elements, BOM handles browser features.

  • Q3: Difference between append() and appendChild()?
    Ans: append() accepts text and multiple nodes, appendChild() accepts only nodes.

  • Q4: What is classList?
    Ans: classList is used to manage CSS classes dynamically.

  • Q5: How do you remove an element?
    Ans: Using remove() or removeChild().
