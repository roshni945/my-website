🐍 What is OOP?

OOP stands for Object-Oriented Programming. It is a programming approach that uses classes and objects to organize code.

✨ Why Use OOP?

  • Code becomes reusable
  • Easy to manage large programs
  • Improves code structure
  • Provides security through encapsulation

📦 Class and Object

A class is a blueprint for creating objects. An object is an instance of a class. 

class Student:

    name = "Roshni"
    age = 18


student1 = Student()

print(student1.name)

🔹 Constructor (__init__ method)

Constructor is a special method that runs automatically when an object is created. 

class Student:

    def __init__(self,name):
        self.name = name


student = Student("Roshni")

print(student.name)

🔗 Inheritance

Inheritance allows one class to use properties and methods of another class. 

class Animal:

    def sound(self):
        print("Animal makes sound")


class Dog(Animal):
    pass


dog = Dog()

dog.sound()

🔄 Polymorphism

Polymorphism means one method can perform different tasks. 

class Cat:

    def sound(self):
        print("Meow")


class Dog:

    def sound(self):
        print("Bark")

🔒 Encapsulation

Encapsulation means wrapping data and methods into a single unit and protecting data. 

class Account:

    def __init__(self):
        self.__balance = 1000

❓ Frequently Asked Questions

1. Is OOP important in Python?

Yes, OOP is widely used for creating large and professional applications.

2. What are the main concepts of OOP?

Class, Object, Inheritance, Polymorphism and Encapsulation are the main concepts.

3. Is Python fully object-oriented?

Yes, Python supports object-oriented programming.
