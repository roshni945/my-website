📦 What are Python Modules?

A module is a file containing Python code such as functions, variables, and classes. Modules help organize large programs into smaller reusable parts.

✨ Why Use Modules?

  • Code reusability
  • Better program organization
  • Easy debugging
  • Saves development time

🔹 Types of Python Modules

  • Built-in Modules - Modules already available in Python. Example: math, random, datetime
  • User-defined Modules - Modules created by programmers.
  • External Modules - Modules installed using pip.

📥 Importing Modules

import math

print(math.sqrt(25))

🔢 Built-in Modules Examples

Math Module

import math

print(math.pi)
print(math.factorial(5))

Random Module

import random

number = random.randint(1,10)

print(number)

DateTime Module

import datetime

today = datetime.datetime.now()

print(today)

🛠️ Creating Your Own Module

Create a Python file and store your functions inside it. Then import that file into another program. 

# mymodule.py

def greet():
    print("Hello Roshni")


# main.py

import mymodule

mymodule.greet()
