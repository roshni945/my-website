    📖 Introduction

    JavaScript Arrays are used to store multiple values in a single variable. In this guide, you'll learn how to create arrays, access elements, use array methods, and loop through arrays with easy examples.

    📋 What is an Array?

    An array stores multiple values in a single variable.

    💡 Did You Know?

    JavaScript Arrays can store different data types together, such as numbers, strings, booleans, objects, and even other arrays.

    📌 Create an Array

    Example:

    let fruits = ["Apple", "Banana", "Mango"];
    Output: ["Apple", "Banana", "Mango"]

    📌 Access Array Elements

    Example:

    console.log(fruits[0]);
console.log(fruits[1]);
    Output:
    Apple
    Banana

    Array indexing starts from 0.

    📌 Array Methods

    Example:

    fruits.push("Orange");
fruits.pop();

console.log(fruits);
    Output: ["Apple", "Banana", "Mango"]

    push() adds an item and pop() removes the last item.

    📊 Common Array Methods

    Method Purpose
    push() Adds an element at the end
    pop() Removes the last element
    shift() Removes the first element
    unshift() Adds an element at the beginning
    length Returns the number of elements

    📌 Loop Through an Array

    Example:

    for(let i = 0; i fruits.length; i++){
    console.log(fruits[i]);
}
    Output:
    Apple
    Banana
    Mango

    ⭐ Key Points

    • Arrays store multiple values in one variable.
    • Array indexing starts from 0.
    • Use push() to add elements.
    • Use pop() to remove the last element.
    • Use loops to access array elements.

    ⭐ Best Practices

    • Use meaningful array names.
    • Use loops for large arrays.
    • Use array methods whenever possible.
    • Avoid duplicate values unless required.
