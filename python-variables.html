📘 What are Python Variables?

A Python Variable is a name used to store data in memory. Variables help us save values such as numbers, text, lists and more, so we can use them later in our program.

Unlike many programming languages, Python does not require you to declare the data type before creating a variable. Python automatically detects the type based on the assigned value.

📝 Python Variable Syntax

The basic syntax is:


variable_name = value

Example:


name = "Roshni"
age = 19
marks = 89.5

Here,

  • name stores a string.
  • age stores an integer.
  • marks stores a floating-point number.

💻 First Python Variable Example


name = "Roshni"

print(name)

Output:


Roshni

In this example, the variable name stores the value "Roshni". The print() function displays the value on the screen.

📜 Python Variable Naming Rules

Python variables must follow these important rules:

  • ✅ A variable name must start with a letter (A-Z, a-z) or an underscore (_).
  • ✅ It can contain letters, numbers and underscores.
  • ❌ It cannot start with a number.
  • ❌ Spaces are not allowed.
  • ❌ Special characters like @, #, %, ! are not allowed.
  • ✅ Variable names are case-sensitive.

✅ Valid Variable Names


name = "Roshni"

student_name = "Aman"

age2 = 19

_marks = 95

❌ Invalid Variable Names


2name = "Roshni"

student name = "Aman"

my-name = "ABC"

@marks = 95

Why?

  • Starts with a number ❌
  • Contains spaces ❌
  • Contains hyphen (-) ❌
  • Contains special characters ❌

💡 Important Notes

  • Python automatically detects the data type.
  • You don't need to declare variables before using them.
  • Always use meaningful variable names.
  • Follow snake_case naming style for better readability.

🔄 Assign Multiple Variables

Python allows you to assign values to multiple variables in a single line. This makes your code shorter and cleaner. 


x, y, z = 10, 20, 30

print(x)
print(y)
print(z)

Output:


10
20
30

📦 Assign Same Value to Multiple Variables

You can also assign the same value to multiple variables at once. 


x = y = z = 100

print(x)
print(y)
print(z)

Output:


100
100
100

✏️ Change Variable Value

Variables are not permanent. You can change their value anytime. 


name = "Roshni"

print(name)

name = "Python"

print(name)

Output:


Roshni
Python

🧠 Memory Tip

Think of a variable as a labeled storage box. You can store a value inside it, remove it, or replace it with another value whenever needed.

⚠️ Common Mistakes

  • ❌ Starting variable names with numbers.
  • ❌ Using spaces in variable names.
  • ❌ Using Python keywords as variable names.
  • ❌ Using unclear names like a, b, x1 everywhere.

🖨️ Output Variables using print()

The print() function is used to display the value of a variable on the screen. 


name = "Roshni"

print(name)

Output:


Roshni

📌 Print Multiple Variables

You can print multiple variables in one line by separating them with commas. 


name = "Roshni"
age = 19

print(name, age)

Output:


Roshni 19

🔗 Concatenate Variables

Strings can be joined using the + operator. 


first_name = "Roshni"
last_name = "Prajapati"

print(first_name + " " + last_name)

Output:


Roshni Prajapati

💡 Quick Tip

Use meaningful variable names like student_name, total_marks, or user_age instead of x, y, and z. It makes your code easier to understand.
