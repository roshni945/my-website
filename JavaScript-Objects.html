    📖 Glossary

    Object
    A collection of key-value pairs used to store related data.
    Property
    A key and its value inside an object.
    Key
    The name used to identify a value in an object.
    Value
    The data stored for a specific key.
    Dot Notation
    Accessing properties using the dot (.) operator.
    Bracket Notation
    Accessing properties using square brackets [].
    Object.keys()
    Returns an array of all object keys.
    Object.values()
    Returns an array of all object values.
    Object.entries()
    Returns an array of key-value pairs.
    for...in Loop
    Used to iterate through object properties.

    ⚡ Common Object Methods Example

    Example:

    let person = {
    name: "Roshni",
    age: 18
};

console.log(Object.keys(person));
console.log(Object.values(person));
console.log(Object.entries(person));
    Output:
    ["name", "age"]
    ["Roshni", 18]
    [["name","Roshni"],["age",18]]

    💡 Did You Know?

    📖 Introduction

    JavaScript Objects are used to store data as key-value pairs. In this guide, you'll learn how to create objects, access properties, update values, add and delete properties, and loop through objects with easy examples.

    📦 What is an Object?

    A JavaScript object stores data in the form of key-value pairs.

    📝 Create an Object

    Example:

    let person = {
    name: "Roshni",
    age: 18,
    city: "Mumbai"
};
    Object Created Successfully

    🔑 Access Object Properties

    Example:

    console.log(person.name);
console.log(person.age);
    Output:
    Roshni
    18

    ✏️ Update Object Values

    Example:

    person.city = "Pune";

console.log(person.city);
    Output:
    Pune

    ➕ Add New Property

    Example:

    person.country = "India";

console.log(person.country);
    Output:
    India

    ❌ Delete Property

    Example:

    delete person.age;

console.log(person);
    Output:
    { name: "Roshni", city: "Pune", country: "India" }

    🔄 Loop Through Object

    Example:

    for(let key in person){
    console.log(key + " : " + person[key]);
}
    Output:
    name : Roshni
    city : Pune
    country : India

    🛠️ Common Object Methods

    Method Description
    Object.keys() Returns all object keys.
    Object.values() Returns all object values.
    Object.entries() Returns key-value pairs.
    hasOwnProperty() Checks if a property exists.

    ⭐ Key Points

    • Objects store data as key-value pairs.
    • Use dot (.) or bracket [] notation to access properties.
    • Properties can be added, updated, or deleted.
    • Use for...in loop to iterate over objects.
    • Objects are useful for storing related information.

    ⭐ Best Practices

    • Use meaningful property names.
    • Keep related data inside one object.
    • Avoid duplicate property names.
    • Use const when the object reference doesn't change.

    ❓ Frequently Asked Questions

    1. What is an object in JavaScript?

    A JavaScript object is a collection of key-value pairs used to store related data.

    2. How do you create an object?

    You create an object using curly braces { }.

    3. How do you access object properties?

    You can access properties using dot notation (.) or bracket notation ([]).

    4. How do you add a new property?

    Simply assign a new key and value, for example: person.city = "Mumbai";

    5. How do you delete a property?

    Use the delete keyword, for example: delete person.age;
