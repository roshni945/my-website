📖 Introduction

JavaScript Numbers are used to store integers and decimal values. They are useful for calculations, prices, marks, percentages, and many other mathematical operations.

📖 What are JavaScript Numbers?

A Number is a data type used to store numeric values. Numbers can be integers or decimal values. 

let age = 18;
let price = 99.99;

console.log(age);
console.log(price);
Output:
18
99.99

📝 Create Numbers

You can create numbers by assigning numeric values to variables. 

let x = 100;
let y = 50.5;

console.log(x);
console.log(y);
Output:
100
50.5

➕ Number Operations

JavaScript supports addition, subtraction, multiplication and division. 

let a = 20;
let b = 5;

console.log(a + b);
console.log(a - b);
console.log(a * b);
console.log(a / b);
Output:
25
15
100
4

🛠️ Number Methods

toFixed()

let num = 12.456;

console.log(num.toFixed(2));
Output:
12.46

toString()

let num = 100;

console.log(num.toString());
Output:
100

parseInt()

console.log(parseInt("25.75"));
Output:
25

parseFloat()

console.log(parseFloat("25.75"));
Output:
25.75

🧮 JavaScript Math Object

The Math object provides useful mathematical functions and constants.

Example:

console.log(Math.round(4.7));
console.log(Math.floor(4.9));
console.log(Math.ceil(4.1));
console.log(Math.max(10,20,30));
console.log(Math.min(10,20,30));
console.log(Math.random());
Output:
5
4
5
30
10
0.56 (Random Number)

🌍 Real-Life Example

Numbers are used everywhere in websites and applications. 

let price = 799;
let quantity = 3;

let total = price * quantity;

console.log(total);
Output:
2397

⭐ Common Number Methods

Method Description
toFixed() Rounds to fixed decimal places.
toPrecision() Formats with specified digits.
Number() Converts value to number.
parseInt() Converts string to integer.
parseFloat() Converts string to decimal number.
isNaN() Checks whether value is NaN.
Math.round() Rounds to nearest integer.
Math.floor() Rounds downward.
Math.ceil() Rounds upward.
Math.random() Generates random number.

❓ Frequently Asked Questions

1. What is a Number?

A Number stores numeric values like 10, 50.5 and -25.

2. How do you round a number?

Use Math.round().

3. How do you convert a string to a number?

Use Number(), parseInt() or parseFloat().

4. What does Math.random() return?

It returns a random decimal number between 0 and 1.

5. What is the difference between parseInt() and parseFloat()?

parseInt() returns an integer, while parseFloat() returns a decimal number.
