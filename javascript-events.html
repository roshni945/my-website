📖 What are JavaScript Events?

JavaScript Events are actions that happen in a webpage. When a user clicks a button, moves the mouse, presses a key, or submits a form, an event occurs.

JavaScript uses events to make webpages interactive and dynamic.

⚡ Types of JavaScript Events

Common JavaScript events are:

  • Mouse Events
  • Keyboard Events
  • Form Events
  • Window Events
  • Clipboard Events

🎯 addEventListener()

The addEventListener() method is the modern way to handle JavaScript events. It allows us to attach events without writing HTML attributes.
Output:
Event runs when the button is clicked.

📌 Event Object

The event object contains information about the event that happened. 


document.addEventListener("click", function(event){

console.log(event);

});
Shows details about the clicked event.

🗑️ Remove Event Listener

removeEventListener() removes an event that was added using addEventListener(). 


function message(){

alert("Hello");

}


button.addEventListener(
"click",
message
);


button.removeEventListener(
"click",
message
);
The click event is removed.

📖 JavaScript Events Glossary

Event
An action performed by the user or browser, like click or typing.
Event Handler
A function that runs when an event occurs.
Event Listener
A method that waits for an event and executes code.
addEventListener()
A method used to attach events to HTML elements.
Event Object
An object that contains information about an event.
Mouse Event
Events caused by mouse actions like click and hover.
Keyboard Event
Events generated by keyboard actions.
Form Event
Events related to forms like submit and change.
Event Bubbling
A process where an event moves from child element to parent element.
Event Capturing
A process where an event moves from parent element to child element.

❓ Frequently Asked Questions (FAQs)

1. What are JavaScript Events?

JavaScript Events are actions that happen on a webpage such as clicking, typing, or loading a page.

2. Why do we use events in JavaScript?

Events make websites interactive and allow JavaScript to respond to users.

3. What is onclick event?

onclick runs when a user clicks on an HTML element.

4. What is addEventListener()?

It is a modern method used to attach events to elements.

6. What are mouse events?

7. What are keyboard events?

Keyboard events occur when users press or release keyboard keys.

8. What is event object?

Event object provides details about the event that occurred.

9. What is event bubbling?

Event bubbling means an event moves from the child element to its parent.

10. Can JavaScript handle multiple events?

Yes, JavaScript can handle multiple events using addEventListener().
