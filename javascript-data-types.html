📦 What are JavaScript Data Types?

Data Types tell JavaScript what kind of value a variable stores, such as text, numbers, true/false values, objects, or arrays.

🔤 String

let name = "Shreya";

A String stores text.

🔢 Number

let age = 18;

Numbers store integer and decimal values.

✅ Boolean

let isStudent = true;

Boolean has only two values: true and false.

❓ Undefined

let city;
console.log(city);

A variable that has no value is undefined.

⭕ Null

let phone = null;

Null means an empty or intentionally unknown value.

📦 Object

let student = {
  name: "Shreya",
  age: 18
};

Objects store data in key-value pairs.

📋 Array

let colors = ["Red", "Blue", "Green"];

Arrays store multiple values in a single variable.

⭐ Summary

  • String → Text
  • Number → Numbers
  • Boolean → True / False
  • Undefined → No value assigned
  • Null → Empty value
  • Object → Key-value data
  • Array → Collection of values

❓ Frequently Asked Questions

1. How many basic data types are there?

JavaScript has String, Number, Boolean, Undefined, Null, Object and more.

2. Is Array a data type?

Yes. Arrays are special objects used to store multiple values.

3. Which data type stores text?

String stores text values.
