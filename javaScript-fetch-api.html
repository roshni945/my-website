📖 Introduction

The Fetch API is one of the most important features of modern JavaScript. It allows developers to communicate with servers, fetch data from APIs, and create dynamic web applications without refreshing the page.

📖 What is Fetch API?

The Fetch API is a modern JavaScript feature that is used to send and receive data from a server. It allows websites to communicate with external services and APIs.

📘 Definition

The Fetch API provides a JavaScript method called fetch() that is used to make HTTP requests and handle responses asynchronously.

💡 Did You Know?

Fetch API was introduced in modern browsers to replace the older XMLHttpRequest method. It provides cleaner and simpler syntax.

⭐ Why Use Fetch API?

  • ✅ Get data from web servers.
  • ✅ Send user data to servers.
  • ✅ Create dynamic websites.
  • ✅ Build real-world applications.
  • ✅ Work with REST APIs.

🌐 What is API?

API stands for Application Programming Interface. It allows two different applications to communicate and exchange data.

📘 Definition

An API is a bridge that allows a client application to request data from a server and receive a response.

🌍 Real-Life Example

Imagine a restaurant:

  • 👤 Customer → Client
  • 📝 Waiter → API
  • 👨‍🍳 Kitchen → Server
  • 🍽️ Food → Response Data

The waiter connects the customer and kitchen just like an API connects a website and server.

⚙️ How Fetch API Works

When we use the fetch() method, JavaScript sends a request to the server. The server processes the request and returns data.

Fetch API Working Process

User opens website


        ↓


Step 2:

JavaScript calls fetch()


        ↓

        ↓


Step 4:

Server sends response


        ↓


Step 5:

JavaScript receives data


        ↓


Step 6:

Data displayed on webpage

🚀 Simple Fetch Example

Basic fetch() 




fetch("https://jsonplaceholder.typicode.com/users")


.then(response => {


return response.json();


})


.then(data => {


console.log(data);


})


.catch(error => {


console.log(error);


});

✅ Output

User data received from API.

⭐ Key Points

  • ✅ Fetch API is used for server communication.
  • ✅ API works as a bridge between client and server.
  • ✅ fetch() returns a Promise.
  • ✅ Response data is usually received in JSON format.
  • ✅ Fetch API works asynchronously.

📝 Fetch API Syntax

The fetch() method is used to create a network request. It takes a URL as an argument and returns a Promise.

📘 Definition

The fetch() function returns a Promise that contains the response received from the server.

📝 Basic Syntax

fetch() Syntax 




fetch("URL")

.then(response => {


    return response;


})


.then(data => {


    console.log(data);


})


.catch(error => {


    console.log(error);


});

🔍 Syntax Explanation

Part Purpose
fetch() Sends request to server
then() Handles successful response
catch() Handles errors

⚡ Fetch API with Promise

Fetch API uses Promises to handle asynchronous operations. A Promise represents a future result of an operation.

📘 Promise States

  • ⏳ Pending - Operation is running
  • ✅ Fulfilled - Operation completed successfully
  • ❌ Rejected - Operation failed

Example: Promise Handling

Fetch Promise Example 




let request = fetch(
"https://jsonplaceholder.typicode.com/posts"
);



request.then(function(response){


console.log(response);


});

✅ Output

Response Object displayed in console.

📦 Fetch Response Object

When a server sends data back, Fetch API provides a Response Object. It contains information about the server response.

📘 Definition

The Response Object represents the response received from a server after making a fetch request.

⭐ Important Response Properties

Property Description
status Returns HTTP status code
ok Checks if request was successful
headers Contains response headers
url Returns requested URL

Example: Check Response




fetch(
"https://jsonplaceholder.typicode.com/users"
)


.then(response => {


console.log(response.status);


console.log(response.ok);


});

✅ Output

200
true

📄 Converting Response into JSON

Most APIs send data in JSON format. To read this data, we use the response.json() method.

📘 Definition

The response.json() method converts the server response into a JavaScript object.

Example: JSON Conversion

response.json() 




fetch(
"https://jsonplaceholder.typicode.com/users"
)


.then(response => {


return response.json();


})


.then(data => {


console.log(data);


});

✅ Output

JavaScript Object containing user data.

⭐ Key Points

  • ✅ fetch() returns a Promise.
  • ✅ then() handles successful responses.
  • ✅ catch() handles errors.
  • ✅ Response Object contains server information.
  • ✅ response.json() converts data into JavaScript objects.

📡 HTTP Methods in Fetch API

HTTP Methods define what type of action we want to perform with server data. Fetch API supports different HTTP request methods.

📘 Definition

HTTP Method tells the server what operation should be performed on the requested resource.

📊 Common HTTP Methods

Method Purpose
GET Retrieve data from server
POST Send new data to server
PUT Update existing data
DELETE Delete data from server

📥 GET Request using Fetch

GET request is used to fetch or retrieve data from a server.

GET Request Example 




fetch(
"https://jsonplaceholder.typicode.com/users"
)


.then(response => response.json())


.then(data => {


console.log(data);


});

✅ Output

User data received from server.

📤 POST Request using Fetch

POST request is used to send new data to the server.

POST Request Example 




fetch(
"https://jsonplaceholder.typicode.com/posts",
{


method:"POST",


headers:{


"Content-Type":"application/json"


},


body:JSON.stringify({


title:"JavaScript",

body:"Fetch API",

userId:1


})


}


)


.then(response => response.json())


.then(data => {


console.log(data);


});

✅ Output

New data sent successfully.

⚡ Async/Await with Fetch API

Modern JavaScript uses async/await to handle Fetch API requests. It makes asynchronous code easier to read and understand.

📘 Definition

async/await is a modern JavaScript feature that allows Promise-based code to be written in a simple and synchronous-looking way.

📝 Async/Await Syntax

Fetch with Async/Await 




async function getData(){


let response = await fetch("API_URL");


let data = await response.json();


console.log(data);


}


getData();

🚀 Example: Fetch Data




async function getUsers(){


const response = await fetch(
"https://jsonplaceholder.typicode.com/users"
);


const users = await response.json();


console.log(users);


}


getUsers();

✅ Output

User information displayed in console.

⚠️ Error Handling in Fetch API

Errors can happen because of wrong URLs, server problems, or internet connection issues. We use try...catch to handle these errors.

📘 Definition

Error handling allows developers to manage unexpected problems without crashing the application.

Example: Try Catch with Fetch

Fetch Error Handling 




async function getData(){


try{


const response = await fetch(
"https://example.com/data"
);



if(!response.ok){


throw new Error("Data not found");


}



const data = await response.json();


console.log(data);



}


catch(error){


console.log(error.message);


}


}



getData();

✅ Output

Data not found

💼 Fetch API Interview Questions

Q1. What is Fetch API?

Fetch API is a JavaScript interface used to make HTTP requests and communicate with servers.

Q2. What does fetch() return?

The fetch() method returns a Promise.

Q3. Why do we use response.json()?

It converts the server response into a JavaScript object.

Q4. Difference between Fetch API and XMLHttpRequest?

Fetch API provides cleaner syntax and uses Promises, while XMLHttpRequest uses older callback-based syntax.

Q5. How to handle Fetch API errors?

Errors can be handled using catch() or try...catch with async/await.

❓ Frequently Asked Questions (FAQs)

Q1. Is Fetch API asynchronous?

Yes, Fetch API works asynchronously and does not block other code.

Q2. Can Fetch API send data?

Yes, Fetch API can send data using POST, PUT, and DELETE methods.

Q3. What format does API usually return?

Most APIs return data in JSON format.

Q4. Is Fetch API supported by modern browsers?

Yes, all modern browsers support Fetch API.
