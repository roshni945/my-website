📖 Introduction

JavaScript Strings are used to store and manipulate text. A string can contain letters, numbers, symbols, spaces, and special characters. JavaScript provides many built-in methods to work with strings easily.

📖 What is a String?

A String is a sequence of characters enclosed in quotes. You can use single quotes (' '), double quotes (" "), or backticks (` `).

Example:

let text = "Hello JavaScript";

console.log(text);
Output:
Hello JavaScript

📝 Create a String

You can create strings in three different ways.

let name = "Roshni";

let city = 'Mumbai';

let message = `Welcome`;
Output:
Roshni
Mumbai
Welcome

📏 String Length

let text = "JavaScript";

console.log(text.length);
Output:
10

🔑 Access Characters

let text = "JavaScript";

console.log(text[0]);

console.log(text.charAt(4));

console.log(text.at(-1));
Output:
J
S
t

➕ Concatenate Strings

let first = "Hello";

let second = "World";

console.log(first + " " + second);

console.log(first.concat(" ", second));
Output:
Hello World
Hello World

🔄 Change Case

let text = "JavaScript";

console.log(text.toUpperCase());

console.log(text.toLowerCase());
Output:
JAVASCRIPT
javascript

✂️ Slice() & Substring()

let text = "JavaScript";

console.log(text.slice(0,4));

console.log(text.substring(4));
Output:
Java
Script

🛠️ Common String Methods

Method Description
length Returns the length of a string.
charAt() Returns the character at a specific index.
at() Returns the character at the given index (supports negative index).
concat() Joins two or more strings.
toUpperCase() Converts text to uppercase.
toLowerCase() Converts text to lowercase.
slice() Extracts part of a string.
substring() Returns part of a string.
replace() Replaces text.
trim() Removes extra spaces.
includes() Checks whether text exists.
indexOf() Returns the first position of a value.
startsWith() Checks whether the string starts with a value.
endsWith() Checks whether the string ends with a value.
split() Splits a string into an array.

💡 Did You Know?

Every text enclosed in quotes is a string. Even "123" is a string, not a number.
