📖 Introduction

The DOM (Document Object Model) is one of the most important concepts in JavaScript. It allows JavaScript to access, modify, create, delete, and update HTML elements dynamically. Every modern website uses DOM Manipulation to build interactive user interfaces.

📖 What is DOM?

DOM (Document Object Model) is a programming interface for HTML documents. It represents every HTML element as an object, allowing JavaScript to access, modify, create, update, and delete webpage content dynamically.

📘 Definition

The Document Object Model (DOM) is a tree-like structure created by the browser after loading an HTML document. JavaScript uses this structure to interact with web page elements.

💡 Did You Know?

The DOM is created automatically by the browser whenever an HTML page is loaded. JavaScript does not create the DOM—it simply interacts with it.

🌳 How DOM Works

When a browser reads an HTML document, it converts every HTML tag into an object. These objects are connected together like branches of a tree, known as the DOM Tree.

Example 1: Access an HTML Element

<h1 id="title">Welcome</h1>

<script>

const heading = document.getElementById("title");

console.log(heading);

</script>

✅ Output

<h1 id="title">Welcome</h1>

Example 2: Change Text Using DOM

<h2 id="msg">Hello</h2>

<script>

document.getElementById("msg").innerHTML = "Welcome to Roshni Code Charm";

</script>

✅ Output

Welcome to Roshni Code Charm

🌍 Real-Life Example

Whenever you click the ❤️ Like button on Instagram, submit a form, add a product to a shopping cart, or switch between Light Mode and Dark Mode, JavaScript updates the webpage instantly using the DOM without refreshing the entire page.

⭐ Key Points

  • ✅ DOM stands for Document Object Model.
  • ✅ Every HTML element becomes a JavaScript object.
  • ✅ JavaScript uses the DOM to access and modify webpages.
  • ✅ The browser automatically creates the DOM.
  • ✅ DOM makes websites interactive and dynamic.

🌳 DOM Tree Structure

When a browser loads an HTML document, it converts every HTML tag into a Node (Object). These nodes are connected in a hierarchical structure called the DOM Tree.

📘 Definition

A DOM Tree is a tree-like representation of an HTML document where every HTML element becomes a node. The document object is the root of the tree.

💡 Did You Know?

The browser automatically creates the DOM Tree every time a webpage is loaded. JavaScript then uses this tree to access and modify HTML elements.

🌲 Example HTML

Example 1: HTML Document

<!DOCTYPE html>
<html>

    <head>
        <title>DOM Example</title>
    </head>

    <body>

        <h1>Welcome</h1>

        <p>Learning DOM</p>

    </body>

</html>

🌳 DOM Tree Representation

document

│

└── html

    ├── head

    │   └── title

    │       └── "DOM Example"

    │

    └── body

        ├── h1

        │   └── "Welcome"

        │

        └── p

            └── "Learning DOM"

🌍 Real-Life Example

Think of the DOM Tree like a family tree.

  • 🌍 document is the Grandparent.
  • 🏠 html is the Parent.
  • 📄 head and body are Children.
  • 📝 h1, p, div, img are Grandchildren.

JavaScript can move through this family tree to find any HTML element.

⭐ Key Points

  • ✅ Every HTML element becomes a DOM Node.
  • ✅ document is the root object.
  • ✅ The browser creates the DOM Tree automatically.
  • ✅ JavaScript uses the DOM Tree to access and modify elements.
  • ✅ Parent, Child, and Sibling relationships exist between nodes.

🌐 Browser Object Model (BOM) vs DOM

Many beginners confuse the Browser Object Model (BOM) with the Document Object Model (DOM). Although both are used in JavaScript, they have different purposes.

📘 Definition

DOM (Document Object Model) represents the HTML document and allows JavaScript to access and modify webpage elements.

BOM (Browser Object Model) represents the browser window and provides features such as navigation, browser history, screen information, alerts, timers, and location.

💡 Did You Know?

The document object belongs to the DOM, while the window object belongs to the BOM.

📊 DOM vs BOM Comparison

Feature DOM BOM
Meaning Document Object Model Browser Object Model
Represents HTML Document Browser Window
Main Object document window
Used For Modify HTML & CSS Control Browser Features
Examples getElementById() alert(), history, location

Example 2: BOM Example

console.log(location.href);

✅ Output

Displays an alert, browser width, and current page URL.

🌍 Real-Life Example

  • 📝 Changing a paragraph's text → DOM
  • 🎨 Changing a button's color → DOM
  • 🔔 Showing an alert box → BOM
  • 🔄 Redirecting to another page → BOM
  • 📜 Accessing browser history → BOM

⭐ Key Points

  • ✅ DOM works with HTML documents.
  • ✅ BOM works with the browser window.
  • ✅ DOM uses the document object.
  • ✅ BOM uses the window object.
  • ✅ Both are essential for modern JavaScript development.

🔍 JavaScript getElementById()

The getElementById() method is one of the most commonly used DOM methods in JavaScript. It selects an HTML element using its unique id attribute.

📘 Definition

The document.getElementById() method returns the HTML element whose id matches the specified value. Since IDs are unique, it always returns a single element.

💡 Did You Know?

Every HTML page should have only one element with a particular id. Duplicate IDs are considered invalid HTML.

📝 Syntax

Syntax

document.getElementById("idName");

Example 1: Select an Element

<h1 id="title">Hello World</h1>

<script>

const heading = document.getElementById("title");

console.log(heading);

</script>

✅ Output

<h1 id="title">Hello World</h1>

Example 2: Change Text

<p id="demo">Welcome</p>

<script>

document.getElementById("demo").innerHTML = "Welcome to Roshni Code Charm";

</script>

✅ Output

Welcome to Roshni Code Charm

Example 3: Change CSS Style

<h2 id="heading">JavaScript DOM</h2>

<script>

document.getElementById("heading").style.color = "blue";

document.getElementById("heading").style.fontSize = "35px";

</script>

✅ Output

The heading becomes blue with a font size of 35px.

Example 4: Change Image

<img id="photo" src="cat.jpg">

<script>

document.getElementById("photo").src = "dog.jpg";

</script>

✅ Output

The displayed image changes from cat.jpg to dog.jpg.

🌍 Real-Life Example

Imagine you click a Login button. JavaScript uses getElementById() to access the username and password fields, validate the input, and display a success or error message without reloading the page.

⭐ Key Points

  • ✅ Selects an element by its unique id.
  • ✅ Returns only one HTML element.
  • ✅ IDs should always be unique on a webpage.
  • ✅ Commonly used to change text, styles, images, and attributes.
  • ✅ One of the most frequently used DOM methods in JavaScript.

🏷️ JavaScript getElementsByClassName()

The getElementsByClassName() method is used to select all HTML elements that have the same class name. Unlike getElementById(), this method returns multiple elements.

📘 Definition

document.getElementsByClassName() returns a live HTMLCollection containing all elements with the specified class name.

💡 Did You Know?

Since multiple HTML elements can have the same class, this method returns an HTMLCollection, not a single element.

📝 Syntax

Syntax

document.getElementsByClassName("className");

Example 1: Select Elements by Class

<p class="demo">Apple</p>

<p class="demo">Banana</p>

<script>

const items = document.getElementsByClassName("demo");

console.log(items);

</script>

✅ Output

HTMLCollection(2)

Example 2: Access First Element

<p class="demo">HTML</p>

<p class="demo">CSS</p>

<script>

const items = document.getElementsByClassName("demo");

console.log(items[0].innerHTML);

</script>

✅ Output

HTML

Example 3: Change All Elements

<p class="demo">One</p>

<p class="demo">Two</p>

<p class="demo">Three</p>

<script>

const items = document.getElementsByClassName("demo");

for(let i = 0; i < items.length; i++){

    items[i].style.color = "blue";

}

</script>

✅ Output

All paragraphs become blue.

Example 4: Change Font Size

<p class="text">JavaScript</p>

<p class="text">DOM</p>

<script>

const text = document.getElementsByClassName("text");

text[0].style.fontSize = "30px";

text[1].style.fontWeight = "bold";

</script>

✅ Output

The first paragraph becomes larger and the second becomes bold.

🌍 Real-Life Example

Suppose an online shopping website has several products marked with the class "product". JavaScript can select all these products using getElementsByClassName() and apply the same style, discount, or event to every product at once.

⭐ Key Points

  • ✅ Selects elements using their class name.
  • ✅ Returns an HTMLCollection.
  • ✅ Multiple elements can share the same class.
  • ✅ Access elements using an index like [0], [1].
  • ✅ Often used with loops to update multiple elements.

📑 JavaScript getElementsByTagName()

📘 Definition

The document.getElementsByTagName() method returns an HTMLCollection containing all elements that match the specified HTML tag name.

💡 Did You Know?

You can select every paragraph, heading, image, button, or any other HTML element simply by using its tag name.

<h2>CSS</h2>

  • ✅ Access elements using an index like [0], [1].

🎯 JavaScript querySelector()

The querySelector() method is one of the most powerful and modern DOM methods in JavaScript. It selects the first HTML element that matches a specified CSS selector.

📘 Definition

The document.querySelector() method returns the first matching HTML element based on a CSS selector such as an ID, class, tag name, or attribute selector.

💡 Did You Know?

querySelector() works with CSS selectors, so you can select elements using IDs (#id), classes (.class), tag names, attributes, and more.

📝 Syntax

Syntax

document.querySelector("CSS Selector");

Example 1: Select by ID

<h1 id="title">Welcome</h1>

<script>

const heading = document.querySelector("#title");

console.log(heading);

</script>

✅ Output

<h1 id="title">Welcome</h1>

Example 2: Select by Class

<p class="demo">HTML</p>

<p class="demo">CSS</p>

<script>

const item = document.querySelector(".demo");

console.log(item.innerHTML);

</script>

✅ Output

HTML

Example 3: Select by Tag Name

<h2>JavaScript</h2>

<h2>DOM</h2>

<script>

const heading = document.querySelector("h2");

heading.style.color = "blue";

</script>

✅ Output

The first heading becomes blue.

Example 4: Select Using Attribute

<input type="text" placeholder="Enter Name">

<script>

const input = document.querySelector("input[type='text']");

input.style.border = "3px solid green";

</script>

✅ Output

The text input gets a green border.

🌍 Real-Life Example

On an online shopping website, the first "Add to Cart" button can be selected using querySelector() and updated when a user clicks it.

⭐ Key Points

  • ✅ Returns only the first matching element.
  • ✅ Uses CSS selectors.
  • ✅ Can select by ID, class, tag name, or attributes.
  • ✅ One of the most commonly used DOM methods.
  • ✅ Preferred in modern JavaScript because it is flexible and powerful.

🎯 JavaScript querySelectorAll()

The querySelectorAll() method is a modern DOM method that selects all HTML elements matching a specified CSS selector. Unlike querySelector(), which returns only the first matching element, querySelectorAll() returns every matching element.

📘 Definition

The document.querySelectorAll() method returns a NodeList containing all HTML elements that match the specified CSS selector.

💡 Did You Know?

querySelectorAll() returns a NodeList, which can be easily looped through using for...of or forEach().

📝 Syntax

Syntax

document.querySelectorAll("CSS Selector");

Example 1: Select All Paragraphs

<p>HTML</p>

<p>CSS</p>

<p>JavaScript</p>

<script>

const paragraphs = document.querySelectorAll("p");

console.log(paragraphs);

</script>

✅ Output

NodeList(3)

Example 2: Select All Elements with a Class

<h2 class="title">HTML</h2>

<h2 class="title">CSS</h2>

<h2 class="title">JavaScript</h2>

<script>

const titles = document.querySelectorAll(".title");

console.log(titles);

</script>

✅ Output

NodeList(3)

Example 3: Change All Paragraph Colors

<p>Apple</p>

<p>Banana</p>

<p>Orange</p>

<script>

const items = document.querySelectorAll("p");

items.forEach(function(item){

    item.style.color = "red";

});

</script>

✅ Output

All paragraphs become red.

Example 4: Change Background Color

<div class="box">One</div>

<div class="box">Two</div>

<div class="box">Three</div>

<script>

const boxes = document.querySelectorAll(".box");

boxes.forEach(function(box){

    box.style.backgroundColor = "lightblue";

});

</script>

✅ Output

All boxes get a light blue background.

🌍 Real-Life Example

Imagine an e-commerce website displaying dozens of products with the same class name. Using querySelectorAll(".product"), JavaScript can instantly update the style, price, or event for every product card at once.

⭐ Key Points

  • ✅ Returns all matching elements.
  • ✅ Uses CSS selectors.
  • ✅ Returns a NodeList.
  • ✅ Supports forEach() and for...of loops.
  • ✅ More flexible than older DOM selection methods.
  • ✅ Widely used in modern JavaScript projects.

⚖️ Difference Between DOM Selection Methods

JavaScript provides several DOM methods to select HTML elements. Each method works differently depending on whether you want to select an element by its ID, Class, Tag Name, or CSS Selector.

📘 Quick Overview

Choosing the correct DOM selector makes your JavaScript code simpler, faster, and easier to understand.

💡 Did You Know?

Modern JavaScript developers mostly use querySelector() and querySelectorAll() because they support CSS selectors.

📊 Comparison Table

Method Selects By Returns Modern?
getElementById() ID Single Element ⭐⭐⭐⭐⭐
getElementsByClassName() Class HTMLCollection ⭐⭐⭐
getElementsByTagName() Tag Name HTMLCollection ⭐⭐⭐
querySelector() CSS Selector First Matching Element ⭐⭐⭐⭐⭐
querySelectorAll() CSS Selector NodeList ⭐⭐⭐⭐⭐

🌍 When Should You Use Which?

  • 🆔 Use getElementById() when selecting one unique element.
  • 🏷️ Use getElementsByClassName() when many elements share the same class.
  • 📑 Use getElementsByTagName() to select all elements of one HTML tag.
  • 🎯 Use querySelector() to select the first matching element with a CSS selector.
  • 🎯 Use querySelectorAll() to select every matching element with a CSS selector.

⭐ Key Differences

  • ✅ getElementById() returns only one element.
  • ✅ getElementsByClassName() returns an HTMLCollection.
  • ✅ getElementsByTagName() returns an HTMLCollection.
  • ✅ querySelector() returns the first matching element.
  • ✅ querySelectorAll() returns a NodeList of all matching elements.
  • ✅ querySelector() and querySelectorAll() support CSS selectors.
  • ✅ Modern developers usually prefer querySelector() and querySelectorAll().

💼 JavaScript DOM Manipulation Part 1 - Interview Questions

These are the most frequently asked JavaScript DOM Manipulation Interview Questions for freshers, internships, BCA/BSc students, and web developer interviews.

1. What is the DOM in JavaScript?

Answer: DOM (Document Object Model) is a programming interface that represents an HTML document as a tree of objects. It allows JavaScript to access and modify webpage content dynamically.

2. What is DOM Manipulation?

Answer: DOM Manipulation is the process of changing HTML elements, CSS styles, attributes, and webpage content using JavaScript.

3. What is the difference between DOM and BOM?

Answer:

  • DOM: Represents the HTML document.
  • BOM: Represents the browser window and browser features.

4. What is the document object?

Answer: The document object represents the current HTML page and is the starting point for all DOM operations.

5. What is getElementById()?

Answer: It selects a single HTML element using its unique ID.

6. What is getElementsByClassName()?

Answer: It selects all elements having the same class name and returns an HTMLCollection.

7. What is getElementsByTagName()?

Answer: It selects all HTML elements with a specified tag name and returns an HTMLCollection.

Answer:

  • querySelector() returns the first matching element.
  • querySelectorAll() returns all matching elements as a NodeList.

9. Which DOM selector is most commonly used in modern JavaScript?

Answer: querySelector() and querySelectorAll() because they support CSS selectors and are more flexible.

10. What is an HTMLCollection?

Answer: HTMLCollection is a collection of HTML elements returned by methods like getElementsByClassName() and getElementsByTagName().

11. What is a NodeList?

Answer: NodeList is a collection of nodes returned by querySelectorAll(). It supports methods like forEach().

12. Why is getElementById() faster than other selectors?

📖 JavaScript DOM Manipulation Part 1 - Glossary

Here are some important terms you learned in this chapter. Understanding these keywords will help you build a strong foundation in JavaScript DOM Manipulation.

🌐 DOM

The Document Object Model represents an HTML page as a tree of objects that JavaScript can access and modify.

🖥️ BOM

The Browser Object Model represents browser features such as the window, history, navigator, and location.

📄 document

The document object represents the current HTML document and is the starting point of DOM Manipulation.

🆔 ID

A unique identifier assigned to an HTML element. It is selected using getElementById().

🏷️ Class

A class groups multiple HTML elements together. It is commonly selected using getElementsByClassName().

📑 Tag Name

The name of an HTML element such as p, h1, div, or img.

🎯 querySelector()

Selects the first HTML element that matches a CSS selector.

🎯 querySelectorAll()

Selects all HTML elements matching a CSS selector and returns a NodeList.

📦 HTMLCollection

A live collection of HTML elements returned by methods like getElementsByClassName() and getElementsByTagName().

📋 NodeList

A collection of nodes returned by querySelectorAll(). It supports methods such as forEach().

🎨 CSS Selector

A pattern used to select HTML elements, such as #id, .class, or div p.

🌳 DOM Tree

The hierarchical structure of an HTML document where every element is represented as a node.

Memorizing these terms will make it much easier to understand future JavaScript topics such as Events, Forms, DOM Traversal, and Dynamic Web Applications.

Here are some commonly asked questions about JavaScript DOM Manipulation Part 1.

1. What is the DOM in JavaScript?

The Document Object Model (DOM) is a programming interface that represents an HTML document as a tree of objects. It allows JavaScript to read, modify, create, and delete HTML elements dynamically.

2. Why is DOM Manipulation important?

DOM Manipulation makes web pages interactive. It allows developers to change text, images, styles, forms, and other webpage elements without reloading the page.

3. What is the difference between DOM and BOM?

DOM represents the HTML document, while BOM (Browser Object Model) represents browser-related objects like window, history, navigator, and location.

4. What does getElementById() return?

It returns a single HTML element whose ID matches the specified value. If no matching element exists, it returns null.

5. What is the difference between querySelector() and querySelectorAll()?

querySelector() returns the first matching element, while querySelectorAll() returns all matching elements as a NodeList.

6. What is an HTMLCollection?

An HTMLCollection is a live collection of HTML elements returned by methods like getElementsByClassName() and getElementsByTagName().

7. What is a NodeList?

A NodeList is a collection of nodes returned by querySelectorAll(). It supports iteration using forEach() and loops.

8. Which DOM selector should I use?

For modern JavaScript, use querySelector() and querySelectorAll(). Use getElementById() when selecting a single element by ID.

9. Can multiple HTML elements have the same ID?

No. Every ID should be unique within an HTML document. If multiple elements have the same ID, the page becomes invalid and may produce unexpected results.
