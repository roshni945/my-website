📝 What are JavaScript Variables?

Variables are containers used to store data in JavaScript. You can store text, numbers, true/false values, arrays, and objects inside variables.

📌 var Keyword

The var keyword was used in older versions of JavaScript. Nowadays, developers usually use let and const. 

var name = "Roshni";
console.log(name);

📌 let Keyword

The let keyword is used for variables whose value can change. 

let age = 18;

age = 19;

console.log(age);

📌 const Keyword

Use const when the value should never change. 

const country = "India";

console.log(country);

📌 Variable Naming Rules

  • Start with a letter, _ or $.
  • Do not start with a number.
  • Variable names are case-sensitive.
  • Avoid using reserved keywords.

💻 Example

let student = "Shreya";
let marks = 95;
const city = "Mumbai";

console.log(student);
console.log(marks);
console.log(city);

⭐ Best Practices

  • Use const whenever possible.
  • Use let if the value changes.
  • Avoid using var in modern JavaScript.
  • Choose meaningful variable names.

❓ Frequently Asked Questions

1. Which keyword is best?

Use const by default and let when the value changes.

2. Can we change a const variable?

No. A const variable cannot be reassigned.

3. Is var still used?

Yes, but modern JavaScript mainly uses let and const.
