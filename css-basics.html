🎨 CSS Basics for Beginners

CSS stands for Cascading Style Sheets. It is used to make websites beautiful by adding colors, layouts, fonts, and designs.

What is CSS?

CSS controls the appearance of HTML elements. With CSS, we can change the background, text color, size, spacing, and overall design of a webpage.

Basic CSS Example

h1 {
  color: purple;
  font-size: 40px;
}

Types of CSS

  • Inline CSS - CSS written inside an HTML tag.
  • Internal CSS - CSS written inside the <style> tag.
  • External CSS - CSS written in a separate .css file.

Common CSS Properties

  • color - Changes text color
  • background-color - Adds background color
  • font-size - Changes text size
  • margin - Adds space outside elements
  • padding - Adds space inside elements

Why Learn CSS?

CSS helps developers create attractive, responsive, and user-friendly websites.

Conclusion

CSS is an important skill for every web developer. After learning HTML, CSS is the next step to create beautiful websites.
